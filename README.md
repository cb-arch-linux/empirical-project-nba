## Project Overview
 
Joel Embiid won the 2022-23 MVP. This project argues the data says otherwise. The central argument is that MVP voters may not have fully accounted for how dominant Nikola Jokić was, instead opting for media narratives and counting stats, such as points per game, to make a decision. The analysis constructs three custom impact metrics adapted from the published Box Plus Minus methodology, trains a ridge regression model on seven seasons of historical MVP data to derive composite score weights without data leakage, runs an OLS regression to test statistical significance of each metric, and applies a causal forest to estimate whether high usage rate genuinely causes better player efficiency.
 
**Verdict: Nikola Jokić (MVP Score: 0.825), ahead of Giannis Antetokounmpo (0.767) and Joel Embiid (0.733).**

## Data Sources
 
All data is sourced from the NBA API via the `nba_api` Python package and is used in accordance with [NBA.com Terms of Use](https://www.nba.com/termsofuse).
 
Two datasets are collected. **Historical data (2015-16 to 2021-22)** is used to train the ridge regression model. **Current season data (2022-23)** is the season being analysed.
 
| Endpoint | Measure Type | Columns Used |
|----------|-------------|--------------|
| `LeagueDashPlayerStats` | Base | PPG, RPG, APG, STL, BLK, TOV, FGA, Win% |

## Methodology
 
### Custom Impact Metrics
 
Three custom metrics are constructed from NBA API rate statistics, adapted from the BPM methodology (Myers, 2014) and the win shares framework (Oliver, 2004).
 
**Custom BPM** estimates net impact per 100 possessions using steal rate, block rate, assist rate, offensive and defensive rebound rates, and true shooting percentage relative to the league average. The net rating blend uses each player's contribution *above their team's baseline* to avoid players from being rewarded for playing with good teammates.
 
**Custom VORP** converts Custom BPM into a cumulative season value against a replacement level of -2.0 points per 100 possessions, scaled by minutes played. The 4800 denominator (48 minutes × 100 games) follows Basketball Reference convention.
 
**Custom WS/48** estimates wins contributed per 48 minutes from offensive and defensive rating differentials relative to league averages, pace adjusted (possesions played).
 
### Qualifying Thresholds
 
Three thresholds are applied before scoring, in short, players on losing teams that make their teams worse will not win MVP:
 
- **Team win percentage ≥ 0.50** — since 1975-76, only three players have won MVP below 0.61, so a floor of 0.50 was set
- **Net rating ≥ -2.0** — players whose teams perform significantly below average with them on court are excluded
- **Usage rate ≥ 0.26** — excludes role players

### Ridge Regression and OLS
 
A ridge regression is trained on the 2015-16 to 2021-22 historical dataset, where each player-season is labelled as MVP calibre based on actual top-3 MVP vote finishers that year. The positive regression coefficients are normalised to total 1.0 and used as composite score weights.
 
A supplementary OLS regression provides p-values and confidence intervals. Key findings:
 
| Metric | Coefficient | p-value | Significant |
|--------|-------------|---------|-------------|
| Custom BPM | +0.043 | <0.001 | Yes |
| Usage % | +0.013 | 0.001 | Yes |
| Team Win % | +0.010 | 0.001 | Yes |
| AST % | -0.026 | <0.001 | Yes |
| OREB % | -0.024 | <0.001 | Yes |
| PIE | +0.011 | 0.077 | **No** |
 
Player Impact Estimate (PIE), the NBA's own composite metric, is the only variable that fails to reach significance (p=0.077), suggesting it does not reliably distinguish MVP calibre seasons in the modern era.
 
Net rating is excluded because custom WS/48 is algebraically derived from the same differentials, introducing multicollinearity. Drives per game, drive FG%, and defensive rim FG% were tested but found statistically insignificant and removed.
 
### Causal Forest
 
A Double Machine Learning Causal Forest (`econml.dml.CausalForestDML`) estimates the heterogeneous causal effect of **usage rate (treatment)** on **true shooting percentage (outcome)** for each player. True shooting percentage is used rather than PIE because PIE has a circularity problem with usage.
 
Controls: true shooting, AST%, OREB%, DREB%, net rating, drives per game, custom BPM. The DML approach residualises both treatment and outcome against these controls before fitting the causal forest.

## Results
 
| Rank | Player | Team | MVP Score |
|------|--------|------|-----------|
| 1 | Nikola Jokić | DEN | 0.825 |
| 2 | Giannis Antetokounmpo | MIL | 0.767 |
| 3 | Joel Embiid | PHI | 0.733 |
| 4 | Luka Dončić | DAL | 0.653 |
| 5 | Ja Morant | MEM | 0.564 |
 
Jokić leads on custom BPM (52.3), true shooting (70.1%), and assist percentage (41.2%). No other player came close on all three dimensions simultaneously. Giannis ranks second primarily due to the highest team win percentage in the top 10 (74.6%). Embiid ranks third despite the highest individual scoring average (33.1 PPG), confirming the project's argument that his case was overvalued by the media due to high-volume counting stats and voter fatigue, consistent with the argument developed in the blog post.
